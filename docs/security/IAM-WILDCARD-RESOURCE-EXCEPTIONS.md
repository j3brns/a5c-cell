# IAM Wildcard Resource Exceptions

The platform default remains no wildcard IAM actions and no wildcard IAM resources.
`Resource: "*"` is allowed only when AWS does not offer a resource ARN for the
action, and only with the narrowest supported condition.

Current approved exceptions:

| Action | Holder | Why wildcard remains | Required guardrail |
| --- | --- | --- | --- |
| `cloudwatch:PutMetricData` | Bridge, Billing Lambdas | CloudWatch metric publish has no resource ARN. | `cloudwatch:namespace` must be `Platform/Bridge` or `Platform/Billing`. |
| `cloudwatch:GetMetricStatistics` | Admin Ops Lambda | CloudWatch metric read has no metric resource ARN. | Read-only diagnostics path only; no supported resource or namespace condition applies to this action. |
| `servicequotas:ListServiceQuotas` | Admin Ops Lambda | The list action has no resource ARN and no action condition key. | Read-only diagnostics path only; caller requests `ServiceCode=bedrock-agentcore`. |
| `bedrock-agentcore:CreateMemory` | Tenant Provisioner Lambda | Create actions run before a memory ARN exists. | Request must include `TenantManaged=true` and the `TenantManaged` tag key. Follow-up memory actions use the memory ARN plus `aws:ResourceTag/TenantManaged=true`. |
| `xray:PutTraceSegments`, `xray:PutTelemetryRecords` | Lambda execution roles with active tracing | X-Ray ingestion actions have no resource ARN. | Lambda execution role only; generated by CDK tracing support. |
| `iam:CreateOpenIDConnectProvider`, `iam:DeleteOpenIDConnectProvider`, `iam:UpdateOpenIDConnectProviderThumbprint`, `iam:AddClientIDToOpenIDConnectProvider`, `iam:RemoveClientIDFromOpenIDConnectProvider` | CDK OIDC provider custom-resource role | IAM OIDC provider create/update/delete actions have no resource ARN before the provider exists. | Identity stack only; target URL and audience are fixed in the custom resource properties and pipeline trust tests. |

`infra/guard/platform-security.guard` is the enforcing source. CDK tests must keep
the synthesized template and this exception register aligned.
